A collection of tests showing the nearest point to the mouse cursor on each of the 2D object types supported in the engine. Also can demonstrate OBB overlap tests.

How to Use
- Run VisualMathUnitTests2D_x64 to start the test
Mouse controls pointer
- F2 -> Toggle between point and OBB for mouse pointer
- F5 -> Re-randomize shapes
- 'Q' and 'E' -> rotate OBB pointer left and right
