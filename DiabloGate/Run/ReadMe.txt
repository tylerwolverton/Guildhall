How to Use
- Run DiabloGate_x64 to start the game. 

Keyboard
- WASD -> move camera
	- Shift can be used to lock camera to move with player
- Left click -> move player
- I, Tab -> toggle inventory
- Left click and drag -> move inventory items
- Esc (while dragging inventory item) -> discard item
- Esc -> Pause game
- F5 -> Reload game (more of a debugging tool)
- ~ -> Open dev console
